# CoinPulse 🚀

A real-time cryptocurrency dashboard that streams live market data and visualizes price movements using modern web technologies.

---

## 🔥 Key Features

* 📡 Real-time crypto price updates using WebSockets
* 📊 Live trade tracking and market insights
* ⚡ Fast and responsive UI
* 🌐 Supports multiple cryptocurrencies
* 📈 Interactive data visualization

---

## 🛠 Tech Stack

* Frontend: Next.js, React, Tailwind CSS
* Backend/Data: WebSockets, CoinGecko API
* Deployment: Vercel
